C - Create new Trip

R - All Trips

GET /api/trips?page=1&perPage=5

U - Update Trip

PUT /api/trips/:id
Update using ThunderClient or Postman, input the parameter that you want to change and the new value as json

D - Delete Trip

DELETE /api/trips/:id
